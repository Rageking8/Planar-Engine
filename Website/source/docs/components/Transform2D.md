# Transform2D

Stores the position, rotation and scale of a 2D object.

## Fields

| Name     | Type    | Description                      |
| -------- | ------- | -------------------------------- |
| Position | Vector2 | Position in 2D space.            |
| Rotation | Float   | Rotation in 2D space in degrees. |
| Scale    | Vector2 | Scale of the 2D object.          |
